# WNBA Player Value Analysis

This project evaluates player performance relative to salary investment using WNBA data.

## Objective
To identify high-impact players delivering strong performance relative to cost, and highlight opportunities for more efficient roster construction.

## Key Insights
- Performance and salary are not always aligned
- A cluster of players delivers above-average impact at below-average salaries
- These profiles represent potential value opportunities for roster strategy

- Tableau (Data Visualization)
